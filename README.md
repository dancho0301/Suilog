# Suilog - 水族館訪問記録アプリ

日本全国の水族館への訪問を記録・管理するiOSアプリです。

## 機能

- **マップビュー**: 日本全国82箇所の水族館をマップ上に表示
- **リストビュー**: 水族館を地域別・訪問状況でソートして一覧表示
- **チェックイン**:
  - 位置情報チェックイン（水族館から1km以内で利用可能）
  - 手動チェックイン（過去の訪問を日付指定で記録）
- **マイ水槽**: 訪問した水族館の魚たちが泳ぐアニメーション表示
- **パスポート**: 訪問履歴の閲覧・編集、写真・メモの管理

## 対応水族館

日本全国82施設に対応しています。

| 地域 | 施設数 |
|------|--------|
| 北海道 | 10 |
| 東北 | 6 |
| 関東 | 13 |
| 中部 | 18 |
| 近畿 | 14 |
| 中国・四国 | 11 |
| 九州・沖縄 | 10 |

## 技術スタック

- **UI**: SwiftUI
- **データ永続化**: SwiftData
- **地図**: MapKit
- **位置情報**: CoreLocation

## 動作環境

- iOS 17.0以上
- Xcode 15.0以上

## ビルド方法

```bash
git clone https://github.com/dancho0301/Suilog.git
cd Suilog
open Suilog.xcodeproj
```

Xcodeでプロジェクトを開き、シミュレータまたは実機でビルド・実行してください。
